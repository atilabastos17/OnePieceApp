# OnePieceApp

App baseado em One Piece, que traz informações relevantes sobre os capítulos do mangá.

Foi utilizado no projeto:
  * Swift 5
  * Storyboards
  * Camada de network própria
  * Informações retiradas da API: https://onepiececover.com
